A free, open-source macOS app that transforms raw screen captures into polished videos with algorithm-driven smart zoom, click effects, and timeline editing.

Download for macOS View on GitHub

Requires macOS 13.0 or later

Algorithm-Driven Smart Zoom

Analyzes mouse movements, click patterns, and UI element boundaries to auto-generate zoom keyframes. No manual work needed.

Professional Timeline

Multi-track editor with Transform, Ripple, Cursor & Keystroke tracks. Full easing curve support for cinematic motion.

Free & Open Source

Apache 2.0 licensed. No subscriptions, no telemetry, no vendor lock‑in. Built with pure Apple frameworks.

Intelligent Recording

Smart Auto-Zoom

Algorithms detect mouse movement patterns, UI element boundaries, and scene transitions. Zoom keyframes are generated automatically — focusing in when you click, pulling out during navigation. Every keyframe is fully editable.

Activity Clustering · Session Analysis · Context-Aware Zoom
Precision Editing

Multi-Track Timeline

Four dedicated tracks — Transform, Ripple, Cursor, Keystroke — each with independent keyframes and easing curves. Add, move, or delete keyframes. Preview changes in real time before export.

Transform · Ripple · Cursor · Keystroke Tracks
Visual Feedback

Click Effects

Every mouse click triggers a configurable ripple animation. Color-coded by type — left clicks, right clicks, double-clicks — each distinguishable at a glance. Adjust color, intensity, and duration.

Left · Right · Double Click · Custom Colors
⌘⇧Z
Keyboard Display

Keystroke Overlays

Keyboard shortcuts are detected and overlaid on screen automatically. When you press ⌘C or ⇧⌘Z, it appears with customizable style, duration, fade timing, and position.

Auto-Detection · Shortcuts Only · Repeat Filtering
Aesthetics

Cursors & Backgrounds

Replace the system cursor with 7 built-in styles at adjustable scale. Add gradient, solid color, or image backgrounds with configurable padding, corner radius, and drop shadow.

7 Cursor Styles · Gradient Presets · Shadow Control

Professional Export

Export to MP4 or MOV with H.264, H.265, ProRes 422, or ProRes 4444. Up to 4K UHD at 24, 30, or 60 fps. Quality presets from Low to Original with real-time file size estimation.

How It Works

01

Record

Select a screen or window. Screenize captures video alongside mouse positions, clicks, and keyboard shortcuts automatically.

02

Edit

Review auto-generated keyframes in the timeline editor. Adjust zoom, effects, cursors, and backgrounds.

03

Export

Choose your codec, resolution, and quality. Render the final video with all effects applied.
